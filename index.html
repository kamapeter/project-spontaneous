FIND THE UNITS

The second feature of the web app is "Find The Units". This Feature was designed knowing that many Students tend to forget the units of physical quantities. To use this feature just Enter the name of the physical quantity (make sure your spellings are correct) and the units will be returned as output.For example entering Pressure would output 'Pascal(pa),mmHg' etc.

FIND THE FORMULAE

The second feature of the web app is "Find The Formulae". This Feature is similar to the "Find The Units" Feature. To use this feature just Enter the name of the physical quantity (make sure your spellings are correct) And the Formulae will be returned as output. For example, entering force would output, F=ma,F=PA etc.
